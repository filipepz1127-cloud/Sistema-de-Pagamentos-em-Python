# Sistema-de-Pagamentos-em-Python
O que o programa faz  Solicita o valor total da compra  Exibe um menu de formas de pagamento  Aplica:  10% de desconto à vista no dinheiro/cheque  5% de desconto à vista no cartão  2x sem juros  3x ou mais com 20% de juros  Mostra o valor final e, se houver parcelamento, o valor das parcelas
